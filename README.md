# NoIP-Updater-IPv6
## IPv6 &amp; IPv4 updates for your No-IP Hosts

### Features:
 * No-IP API
 * IPv4 & IPv6 Support
 * Selectable Adapter
 * Use remote or local address
 * Manual updates
 * Copy to Clipboard
 * Flush DNS
 * Autostart
 * Systray
 * Background Worker *(comming soon)*
